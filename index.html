Política de Privacidade — JGlic

Última atualização: 17/01/2026

O JGlic é um aplicativo criado para auxiliar usuários no registro manual de suas glicemias. Esta Política de Privacidade descreve como tratamos os dados fornecidos pelo usuário.

1. Informações Coletadas

O JGlic não coleta automaticamente nenhuma informação pessoal.

Todas as informações inseridas no aplicativo são digitadas voluntariamente pelo usuário, incluindo:

  • Valores de glicemia (anotações manuais);
  • Datas e horários registrados pelo próprio usuário;
  • Alarmes criados dentro do app.

Nenhum dado é coletado por rastreamento, publicidade ou captura automática de informações.

2. Armazenamento dos Dados

2.1. Localmente no dispositivo

Algumas informações podem ser salvas diretamente no dispositivo do usuário.

2.2. Firebase (Google)

Se o usuário optar por utilizar recursos de backup/sincronização, os dados serão armazenados no Firebase, serviço da Google LLC.

Política de privacidade da Google: https://policies.google.com/privacy

O desenvolvedor do JGlic não tem acesso direto às informações armazenadas no Firebase.

3. Compartilhamento de Dados

O JGlic não compartilha dados com terceiros, exceto quando:

  • O usuário utiliza o Firebase voluntariamente;
  • Houver exigência legal ou ordem judicial.

4. Segurança dos Dados

Utilizamos boas práticas de segurança, mas nenhum sistema é totalmente imune. O usuário deve manter seu dispositivo protegido.

5. Exclusão de Dados

O usuário pode excluir seus registros dentro do aplicativo. Caso utilize o Firebase, a exclusão pode ser feita pela conta Google conectada ao app.

6. Alterações nesta Política

Esta política pode ser atualizada ocasionalmente. A data da última alteração estará sempre visível no topo.
